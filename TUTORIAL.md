# Step-by-Step Tutorial - AFH Consultant Masterclass

This tutorial will walk you through using the AFH Consultant Masterclass from start to finish.

## Tutorial Overview

**Duration**: 30-45 minutes  
**Prerequisites**: None  
**Outcome**: Complete understanding of platform features and curriculum content

---

## Part 1: Getting Started (5 minutes)

### Step 1.1: Install and Launch

```bash
# Navigate to project directory
cd /Users/henryoh/Documents/AFHconsultant_1

# Install dependencies (first time only)
npm install

# Start the server
npm run dev
```

**Expected Result**: You see "Ready" message and can access http://localhost:3000

### Step 1.2: Explore the Home Page

1. Open http://localhost:3000 in your browser
2. Read the curriculum overview
3. Notice the four main sections:
   - Home Inspection Compliance
   - Application Submission
   - Top 100 Practice Questions
   - Resources & Checklists

**Action**: Click on each section card to see what's available

---

## Part 2: Module 1 - Home Inspection (10 minutes)

### Step 2.1: Navigate to Module 1

1. Click "Module 1: Home Inspection Compliance" from home page
2. Or navigate to: http://localhost:3000/home-inspection

### Step 2.2: Study the Regulatory Foundation

1. Read "The Big Three Regulatory Frameworks"
2. Understand:
   - WAC 51-51-0330: Building Code
   - WAC 388-76: DSHS Licensing
   - RCW 64.38.060: HOA Protection

**Take Notes**: Write down key WAC numbers and their purposes

### Step 2.3: Learn Physical Requirements

Scroll through and study:
- **Sleeping Rooms**: Window egress (5.7 sq. ft., 44" sill)
- **Bathrooms**: Grab bars (33-36" height, 250 lb. strength)
- **Ramps**: 1:12 slope, 36" width
- **Fire Safety**: Interconnected alarms, extinguishers

**Practice**: Try explaining one requirement to yourself out loud

### Step 2.4: Review the Checklist

Scroll to "The 'One-and-Done' Consultant Checklist"

**Action**: 
1. Read through all 5 sections
2. Identify which items you'd check for a client
3. Note any items you don't understand (research later)

---

## Part 3: Module 2 - Application Submission (10 minutes)

### Step 3.1: Navigate to Module 2

1. Click "Module 2: Application Submission" from home page
2. Or navigate to: http://localhost:3000/application-submission

### Step 3.2: Understand BAAU Process

Study the timeline:
- **60-Day Correction Window**: Time to fix errors
- **One-Year Expiration**: Application validity period
- **Email Communication**: Primary contact method

**Key Point**: Missing the 60-day deadline voids your application!

### Step 3.3: Learn Pre-Submission Requirements

Review:
- **CEA Form**: 1,000 hours of experience
- **Background Checks**: Who needs them
- **Financial Solvency**: 60 days operating capital
- **Property Documentation**: Deed or landlord letter

**Action**: Make a checklist of documents you'd need

### Step 3.4: Study Application Readiness Audit

Scroll to the 10-point audit checklist

**Practice**: 
1. Go through each point
2. Ask yourself: "Do I have this?"
3. Note any missing items

---

## Part 4: Question Bank (10 minutes)

### Step 4.1: Access Questions

1. Click "Top 100 Practice Questions" from home page
2. Or navigate to: http://localhost:3000/questions

### Step 4.2: Explore Categories

1. Notice the category filter buttons at the top
2. Click "All" to see all questions
3. Click different categories to filter

**Try**: Click "Physical Plant & Safety" - you should see 20 questions

### Step 4.3: Study a Question

1. Find Question #11: "What is the maximum height a window sill can be..."
2. Click "Show Answer"
3. Read:
   - **Answer**: 44 inches
   - **Logic**: Why this height is required
   - **Reference**: WAC 51-51-0330

**Practice**: Try to explain the logic in your own words

### Step 4.4: Study by Category

1. Click "Bathrooms & Grab Bars" category
2. Study 3-5 questions
3. Notice the pattern in answers and references

**Insight**: Questions in same category often reference same WAC sections

## Part 6: Practical Application (5 minutes)

### Step 6.1: Scenario Practice

**Scenario**: A client asks: "What's the minimum window size for a resident bedroom?"

**Your Task**:
1. Find the answer in Module 1
2. Explain it using the logic framework
3. Cite the WAC reference

**Answer Should Include**:
- 5.7 square feet (or 5.0 if at grade)
- 20" wide x 24" high minimum
- WAC 51-51-0330 reference

### Step 6.2: Checklist Practice

**Scenario**: Client has inspection in 3 days

**Your Task**:
1. Go to Resources page
2. Review Pre-Inspection Checklist
3. Identify top 5 most critical items

**Top 5 Should Include**:
- Window measurements
- Lever handles
- Grab bars
- Water temperature
- Smoke alarms

### Step 6.3: Question Practice

**Task**: Answer 5 questions without looking at answers first

1. Go to Questions page
2. Pick 5 questions from different categories
3. Write down your answers
4. Check answers
5. Review logic for any you got wrong

## Troubleshooting Tutorial Issues

### Can't see the application?
- Check terminal for errors
- Verify you're on http://localhost:3000
- Try refreshing the page

### Questions not showing?
- Check browser console (F12)
- Verify you clicked "Show Answer"
- Try a different category

### Can't find specific information?
- Use browser search (Ctrl+F / Cmd+F)
- Check the Resources page for quick references
- Review the Manual for detailed information

## Advanced Tutorial: Using as a Consultant

### Step A.1: Client Onboarding

1. Share the Application Readiness Audit with client
2. Review each point together
3. Identify missing items
4. Create action plan

### Step A.2: Pre-Inspection Walkthrough

1. Use Pre-Inspection Checklist

3. Measure windows, test grab bars
4. Document any issues
5. Create Correction Action Plan if needed

### Step A.3: Question-Based Training

1. Select 10-20 questions relevant to client's situation
2. Review together
3. Explain logic and references
4. Test client's understanding
